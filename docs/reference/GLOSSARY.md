# Glossary

> **Navigation**: [Reference](./README.md) | [Documentation Root](../README.md)

---

## A

**AMM**: Automated Market Maker. A decentralized exchange protocol using algorithmic price determination.

**APAC**: Asia-Pacific region.

---

## B

**BEAM**: Bogdan/Bj√∂rn's Erlang Abstract Machine. The virtual machine that runs Elixir and Erlang code.

---

## C

**CLMM**: Concentrated Liquidity Market Maker. An AMM variant allowing liquidity provision within specific price ranges.

---

## E

**EU**: European Union.

---

## G

**GenServer**: Generic Server. An OTP behavior for implementing stateful server processes.

---

## H

**HSM**: Hardware Security Module. Dedicated hardware for secure key storage and cryptographic operations.

---

## K

**KE**: Kinetic Energy. In physical model, represents market momentum.

---

## M

**Mnesia**: Distributed database management system built into Erlang/OTP.

---

## N

**NIF**: Native Implemented Function. Rust or C code callable from Elixir for performance-critical operations.

---

## O

**OTP**: Open Telecom Platform. Erlang/Elixir framework for fault-tolerant distributed systems.

---

## P

**PDA**: Program Derived Address. Deterministic address generation on Solana blockchain.

**PE**: Potential Energy. In physical model, represents price position relative to equilibrium.

---

## R

**R_exit**: Exit range. Upper price range for converting meme coin to stable coin.

**R_fee**: Fee generation range. Central price range for earning trading fees.

**R_restock**: Restock range. Lower price range for acquiring meme coin.

---

## S

**SOL**: Native cryptocurrency of Solana blockchain, used for transaction fees.

**Supervision tree**: OTP process hierarchy with automatic restart on failure.

---

## T

**TBD**: To Be Determined.

**Tick**: Discrete price point in CLMM (price = 1.0001^tick).

**TradFi**: Traditional Finance. Conventional financial markets and institutions.

---

## U

**US**: United States.
