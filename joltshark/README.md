# joltshark

Rust numerical computation library for Cordial Cantina.

## Overview

joltshark provides high-performance numerical operations for the trading system, including physical modeling calculations based on Riemannian geometry concepts.

## Status

**Phase 0**: Stub implementation. Core functionality to be implemented in future phases.

## Building

```sh
cargo build
cargo test
```

## Integration

This crate is consumed by the `nif` subproject which exposes functions to the Elixir application via Rustler NIFs.
